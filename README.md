# SHOP STOPWATCH OPTIMIZER (店舗売上・最適化計算機)

ゲーム内のカフェ店舗における「メニュー5品」と「従業員（キャラクター）最大10名」の最適な組み合わせを、あなたの所持・育成状況に合わせて爆速で自動計算するWebツールです。
GitHub Pagesにプッシュするだけで、PC・スマホのどちらからでも簡単に利用できます。

## 🚀 主な機能

1. **リアルタイム売上計算**
   - 画面上でメニューやキャラクター、発動中の「日次バフ（主食、果物、特定の原材料など）」を切り替えると、ゲーム内の計算式に基づいた予測時給（/h）がリアルタイムに変動・表示されます。
2. **手持ち状況からの「最強編成」自動最適化（高速・スマート版）**
   - 画面下部の管理パネルで自分の「キャラ所持ON/OFF」と「現在のLv」を設定し、中央のバフを選んでボタンを押すだけ。
   - メニューの条件に合わせてキャラの貢献度を動的にその都度計算し、最も時給が高くなる「メニュー5品 ＋ キャラ最大10人」を一瞬で自動セットします。
3. **🔥 完全総当たりモード（100%精度のゴリ押し計算）**
   - 高速版の足切りアルゴリズムを一切使わず、すべてのメニュー×すべての所持キャラの組み合わせを力技で網羅する超高精度モードです。
   - 数百万〜数千万通りの「激重計算」になりますが、**疑似非同期チャンク処理**を搭載しているため、ブラウザをフリーズさせずに進捗（％）を表示しながら安全に計算を完遂できます。
4. **💡 次に育成すべき「おすすめキャラ」＆素材計算機能**
   - 現在の所持・レベル状況から、「どのキャラのレベルを上げれば最も時給が伸びるか」を現在のレベルからLv5までの**すべての段階**でシミュレーションし、TOP3を算出。
   - 効果が発動する目標レベルと、そこまでに必要な育成素材（夢なき果核）の累計数も自動で計算して教えてくれます。
5. **自動保存＆コード入出力機能 (セーブ/ロード)**
   - ブラウザのLocalStorageを利用し、手持ち状況を変更した瞬間に**自動でデータがバックアップ**されます（画面を閉じても消えません）。
   - 機種変更や手動バックアップ用に、ワンクリックで全データを「暗号化コード」に変換してエクスポート・インポートすることも可能です。

## 📁 構成ファイル

- `index.html` ：UI画面（サイバー・ダークモード風デザイン）
- `main.js` ：画面の初期化、イベント制御、自動保存、各種最適化・診断アルゴリズムの制御
- `optimizer.js` ：売上計算ロジック、および高速最適化アルゴリズム
- `data.js` ：メニュー、キャラクター（スキル）、日次バフグループの各種マスターデータ
- `admin.html` ：**【内部ツール】** ブラウザ上で `data.js` の中身を安全かつ簡単に編集・追加・逆パース読込できる管理者用画面

## 🛠️ 使い方

### 1. 初回の手持ち登録＆自動保存
1. 画面最下部の「**全キャラクター所持・育成管理**」エリアを見ます。
2. 自分が「持っていないキャラ」のチェックを外し、「持っているキャラ」のレベル（Lv1〜5）をゲーム内の状態に合わせます。
3. 変更した内容は自動的にブラウザに保存されるため、次回からはアクセスするだけで復元されます。

### 2. 最強編成の自動計算
1. 現在ゲーム内で発動している「**本日の日次バフ**」があれば、該当するものにチェックを入れます。
2. 通常は「⚡ **所持状況から最強の配置を自動計算する**」ボタンを押せば、数ミリ秒で最高効率の組み合わせが自動セットされます。
3. シナジーの取りこぼしがどうしても不安な場合は、「🔥 **完全総当たりモード**」ボタンを押すことで、時間はかかりますが100%確実な絶対解を出力できます。

### 3. 次の育成ターゲットを決める（おすすめ診断）
1. 手持ちのキャラ状況と本日のバフを正確にセットします。
2. 「💡 **次に育成すべき『おすすめキャラTOP3』を診断する**」ボタンを押します。
3. 数十秒の計算後、「誰を」「レベルいくつまで上げると」「時給がいくら伸びて」「夢なき果核が何個必要か」がランキング形式で表示されます。

> **⚠️ 育成診断機能のご利用に関する注意事項**
> 本機能によるレベル上昇時の効率計算や時給増加量は、現在の編成と日次バフ状況に基づいた**あくまで推定（シミュレーション）**です。
> 実際のレベルアップに使用する貴重な育成資材（夢なき果核など）を消費する際は、将来の編成変更や新キャラ追加の可能性などもよくご考慮いただき、**ご自身でよく確認・納得の上で自己責任にてご活用ください。**

## 📊 技術的な特徴（フリーズ回避と高速化）
- **スマート選考アルゴリズム（高速版）：** 全メニュー（8,568通り）に対して毎回「そのメニュー構成に最も特化したキャラトップ10」を独自の貢献度天秤（重み付け）でその都度計算して一時選考し、その中で総当たりを行うことで、スマホブラウザでも一瞬で最適解を導き出します。
- **疑似非同期チャンク処理（総当たり/診断）：** 莫大なループ計算を回す際、100回〜1000回ごとに `setTimeout` を用いて一瞬だけブラウザのスレッドを解放（隙間を提供）しています。これにより、シングルスレッドのJavaScriptでありながら、画面を一切フリーズさせずに「計算中... (45%)」のようなリアルタイム進捗表示を実現しています。
