# SHOP STOPWATCH OPTIMIZER (店舗売上・最適化計算機)

ゲーム内のカフェ店舗における「メニュー5品」と「従業員（キャラクター）最大10名」の最適な組み合わせを、あなたの所持・育成状況に合わせて爆速で自動計算するWebツールです。
GitHub Pagesにプッシュするだけで、PC・スマホのどちらからでも簡単に利用できます。

## 🚀 主な機能

1. **リアルタイム売上計算**
   - 画面上でメニューやキャラクター、発動中の「日次バフ（主食、果物など）」を切り替えると、ゲーム内の計算式に基づいた予測時給（/h）がリアルタイムに変動・表示されます。
2. **手持ち状況からの「最強編成」自動最適化（一瞬で計算！）**
   - 画面下部の管理パネルで自分の「キャラ所持ON/OFF」と「現在のLv」を設定し、中央のバフを選んでボタンを押すだけ。
   - 何万通りもの組み合わせから、現在のあなたの手持ちで**最も時給が高くなる「メニュー5品 ＋ キャラ最大10人」を裏で自動選出し、上のスロットへ一瞬で自動配置**します。
3. **次に育成すべき「おすすめキャラ」診断機能**
   - 現在の所持・レベル状況から、「どのキャラのレベルを上げれば最も時給が伸びるか」をシミュレーションし、TOP3を算出。
   - 効果が発動する目標レベルと、そこまでに必要な育成素材（夢なき果核）の数も同時に教えてくれます。
4. **自動保存＆コード入出力機能 (セーブ/ロード)**
   - ブラウザのLocalStorageを利用し、手持ち状況を変更した瞬間に**自動でデータが保存**されます（画面を閉じても消えません）。
   - 機種変更やバックアップ用に、ワンクリックで全データを「暗号化コード」に変換してメモ帳等にエクスポート・インポートすることも可能です。

## 📁 構成ファイル

- `index.html` ：UI画面（サイバー・ダークモード風デザイン）
- `main.js` ：画面の初期化、イベント制御、自動保存、育成ポテンシャル計算
- `optimizer.js` ：売上計算ロジック、および高速最適化アルゴリズム
- `data.js` ：メニュー、キャラクター（スキル）、日次バフの各種データ
- `admin.html` ：**【内部ツール】** ブラウザ上で `data.js` の中身を安全かつ簡単に編集・追加できる管理者用画面

## 🛠️ 使い方

### 1. 初回の手持ち登録＆セーブ
1. 画面最下部の「**全キャラクター所持・育成管理**」エリアを見ます。
2. 自分が「持っていないキャラ」のチェックを外し、「持っているキャラ」のレベル（Lv1〜5）をゲーム内の状態に合わせます。
3. 変更した内容は自動的にブラウザに保存されるため、次回からはアクセスするだけで復元されます。
4. （※別の端末にデータを引き継ぎたい場合は、「コード作成」を押して出力された文字列をコピーし、新しい端末で「読込適用」から貼り付けてください。）

### 2. 最強編成の自動計算
1. 現在ゲーム内で発動している「**本日の日次バフ**」があれば、該当するものにチェックを入れます。
2. 「⚡ **所持状況から最強の配置を自動計算する**」ボタンをポチッと押します。
3. 画面上の「従業員スロット」と「メニュー設定」に、最高時給を叩き出す組み合わせが自動でカチッとセットされ、予測時給が表示されます！
4. あとはゲーム内の店舗画面を開き、ツールに表示された通りに配置すれば完了です。

### 3. 次の育成ターゲットを決める（おすすめ診断）
1. 手持ちのキャラ状況と本日のバフを正確にセットします。
2. 「💡 **次に育成すべき『おすすめキャラTOP3』を診断する**」ボタンを押します。
3. 数十秒の計算後、「誰を」「レベルいくつまで上げると」「時給がいくら伸びて」「夢なき果核が何個必要か」がランキング形式で表示されます。

> **⚠️ 育成診断機能のご利用に関する注意事項**
> 本機能によるレベル上昇時の効率計算や時給増加量は、現在の編成と日次バフ状況に基づいた**あくまで推定（シミュレーション）**です。
> 実際のレベルアップに使用する貴重な育成資材（夢なき果核など）を消費する際は、将来の編成変更や新キャラ追加の可能性などもよくご考慮いただき、**ご自身でよく確認・納得の上で自己責任にてご活用ください。**

## 📊 技術的な特徴（爆速ロジックについて）
全てのメニュー（18品から5品＝8,568通り）と全キャラクターの全組み合わせをそのまま総当たりすると計算が重くなり、ブラウザがフリーズする原因になります。
本ツールでは、メニューの組み合わせごとに「この商品に対して単体で最も売上貢献度が高い優秀なキャラトップ10」を評価システムで一時選考し、その精鋭たちの中で総当たりを行うスマートアルゴリズムを採用しています。これにより、スマホブラウザでもフリーズせず一瞬で最適解を導き出すことができます。
